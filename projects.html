Linear Regression

This project demonstrates various linear regression techniques on the Boston housing dataset to predict house prices. It uses K-Fold cross-validation to compute average training and test mean squared errors (MSE) for each model, including Ordinary Least Squares, Ridge (with and without polynomial features), Lasso, and Elastic Net. It also implements both analytical solutions (via the normal equation) and gradient-based methods. By comparing the resulting MSEs, we can see how regularization (Ridge, Lasso, Elastic Net) and polynomial expansions affect model performance. Finally, the code visualizes each model's predictions versus the actual housing prices on a separate train-test split and provides a bar chart summarizing and comparing each model's training and test performance.

Logistic Regression

This project implements logistic regression from scratch on the Breast Cancer Wisconsin dataset. It uses gradient descent to learn the weight parameters that best separate malignant and benign tumors. After training, the model's performance is evaluated by calculating precision, recall, F1 score, and a confusion matrix. The code also visualizes the confusion matrix in a heatmap and plots an ROC curve to show how well the model distinguishes between classes across different probability thresholds, with the AUC giving a one-number summary of this separability. Finally, the project includes hyperparameter tuning to find the best learning rate, number of epochs, and decision threshold, thereby demonstrating how these factors affect model performance.

Support Vector Machines (SVM)

This project implements a Support Vector Machine (SVM) from scratch using gradient descent (and mini-batch gradient descent) to learn the model's parameters. It generates a synthetic 3D dataset, splits it into training and test sets, and trains an SVM to separate two classes (labelled +1 and -1). By computing a hinge-loss–like objective function, it updates the weight vector w and bias b to maximize the margin between classes while minimizing misclassifications. The code then visualizes the trained model's decision boundary and margins in both 2D and 3D space, evaluates its accuracy on the test set, and plots the training loss as it changes over iterations.

Decision Trees

This project builds a custom decision tree classifier from scratch (using concepts similar to ID3), applies it to two different datasets (Iris and Spam), and evaluates performance via 10-fold cross-validation. Specifically, it compares accuracy across varying values of n_min, which represents the minimum number of samples required to continue splitting the tree. By experimenting with different n_min settings, the project explores how limiting the tree's growth affects accuracy on training and test folds. Finally, it visualizes both the resulting decision tree structure and the accuracy trends as n_min changes for each dataset.

K-means Clustering

This project demonstrates a clustering approach using multiple techniques. It implements K-means clustering (with both random and k++ centroid initialization), applying it to synthetic data, an image (for image compression), and a retail dataset (Mall Customers). The k++ initialization is modified to use a fixed random seed for reproducible cluster centers. Finally, it includes an agglomerative clustering (Ward linkage) method to build a dendrogram for hierarchical clustering. This enables a comparison of how different clustering algorithms work and how initial conditions or linkages affect the resulting partitions.

Using CNN to Classify AI Images

This project implements a Convolutional Neural Network (CNN) to distinguish between "real" and "fake" images using Photo-Response Non-Uniformity (PRNU) noise features. It first extracts PRNU residuals (using a library sourced from Dr. Miroslav Goljan at Binghamton University), transforms them via the Fourier transform, normalizes the magnitudes, and trains a binary classifier to detect forgeries. The model architecture uses multiple convolutional layers, pooling, dropout, and data augmentation. Finally, it evaluates the trained model on a test set, displays example predictions, and plots training vs. validation accuracy and loss over epochs.
